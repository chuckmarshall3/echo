# Echo() exercise

Modify [echo.php](echo.php) to print out each argument to the `echo` function on a new line. It should work for an arbitrary number of arguments.

```javascript
echo();
// should print nothing

echo('bla');
// should print
//
// 'bla'

echo('foo', 'bar', 'baz');
// should print
//
// 'foo'
// 'bar'
// 'baz'
```
